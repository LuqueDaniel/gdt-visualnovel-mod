GDT - Visual Novel Mod
==========================================================================
This Game Dev Tycoon mod adds support for topics and researches related to visual novels.

**Version**: 0.1<br />
**Requires API version**: 0.1.x

Features
--------------------------------------------------------------------------
- 9 new topics.
- 13 new researchs.
